# 🎮 Tic-Tac-Toe

A clean and fully tested **console-based Tic-Tac-Toe game** written in **Python**, built with a strong focus on **object-oriented design**, **clean architecture**, and **unit testing**.

This project demonstrates solid **junior-level Python skills** and good engineering habits from the ground up.

---

## 🚀 Key Highlights

- Object-oriented design with clear separation of responsibilities
- Fully tested core game logic using **pytest**
- Complete win & draw detection (rows, columns, diagonals)
- Input validation and graceful error handling
- Clean, readable, and easily extensible codebase

---

## 🧠 What This Project Shows

- Understanding of **OOP principles**
- Ability to design **testable code**
- Clean project structure
- Attention to edge cases and correctness
- Practical use of unit tests (not just theory)

---

## 🧩 Project Structure
```project/
├── board.py      # Game board logic (win, draw, move validation)
├── main.py       # Application entry point
├── menu.py       # Game flow and user interaction
├── player.py     # Player representation
│                  
tests/
├── test_board.py
└── test_player.py 
```

## ▶️ How to Run

```bash
python main.py



Players take turns by entering numbers 1–9 to place their symbol on the board.
```
